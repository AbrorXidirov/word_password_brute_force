# word_password_brute_force

# 🔓 DOCX Password Recovery PRO

Ushbu dastur unutilgan yoki yo'qotilgan Microsoft Word (`.docx`) fayllari parolini tiklash (brute-force) uchun mo'ljallangan. Dastur Python va PyQt5 yordamida yozilgan bo'lib, kompyuterning barcha yadrolaridan (Multiprocessing) 100% foydalangan holda maksimal tezlikda ishlaydi.

## 🚀 Imkoniyatlari (3 xil hujum rejimi)
1. **6 xonali sonlar:** `numbers.txt` fayli orqali (masalan, 000000 dan 999999 gacha).
2. **Shaxsiy lug'at:** Dasturning o'zida maxsus parollarni kiritish va saqlash imkoniyati.
3. **Rockyou.txt:** Mashhur xakerlik lug'ati orqali tekshirish.

---

## 🛠 O'rnatish bo'yicha yo'riqnoma (Windows/Linux/Mac)

Dasturni o'z kompyuteringizda ishga tushirish uchun quyidagi qadamlarni bajaring:

### 1. Python o'rnatilganligini tekshiring
Kompyuteringizda Python (3.7 yoki undan yuqori) o'rnatilgan bo'lishi shart. Uni [python.org](https://www.python.org/downloads/) saytidan yuklab olishingiz mumkin.

### 2. Loyihani yuklab oling
Terminal (CMD) ni oching va loyihani kompyuteringizga ko'chirib oling:
```bash
git clone https://github.com/AbrorXidirov/word_password_brute_force.git
cd word_password_brute_force
```
rockyou.txt hajmi katta bo'lganligi uchun uni internatdan yuklab loyiha papkasiga qo'shing.
